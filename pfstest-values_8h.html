Detailed Description

Box various primitive values for use with assert_that(), is(), arg_that(), etc.

Function Documentation

pfstest_value_t* pfstest_as_hex ( pfstest_value_t *  value)

Wrap unsigned primitive integers to cause them to be printed in hexadecimal in failure messages.

For example,

1 the_uint(19)

will be printed as "the uint 19", but

1 as_hex(the_uint(19))

will be printed as "the uint 0x13".

Warning
Only use this wrapper with unsigned values built into PFSTest. Using it with other values will not work and will probably cause memory corrpution or a segfault.
